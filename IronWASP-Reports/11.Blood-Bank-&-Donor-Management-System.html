Cookie PHPSESSID missing the HttpOnly flag
>>>
Type:Vulnerability
Severity:Medium
Confidence:High
Found By:Passive Analysis

Affected Site:http://localhost/
Affected Url:/11.Blood-Bank-&-Donor-Management-System/bbdms/
Description:
The HttpOnly flag was missing on the cookie: PHPSESSID. This may allow an attacker to get the cookie information using XSS attacks.
Analyzed Response:

The value of the cookie is not protected by HttpOnly flag and hence becomes accessible from JavaScript
